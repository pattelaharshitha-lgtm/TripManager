# Trip Management System

## Overview
A Java Swing-based desktop application for planning trips, managing expenses, and tracking budgets.

## Features
- Create trips
- Manage expenses
- Budget tracking
- Expense splitting
- User profile
- Java Swing GUI

## Technologies Used
- Java
- Java Swing
- OOP

## How to Run
1. Clone the repository.
2. Open the project in VS Code or IntelliJ.
3. Compile the Java files.
4. Run MainFrame.java.
